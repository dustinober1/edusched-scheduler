# EduSched

**EduSched** is a constraint-based scheduling package for educational institutions.

## Documentation

For detailed usage instructions, please refer to the [USER_GUIDE.md](USER_GUIDE.md).

## Installation

```bash
pip install edusched
```

## Features

*   Define resources (Rooms, Teachers)
*   Set constraints (No overlap, Blackout dates, etc.)
*   Define objectives (Spread evenly, Minimize evening sessions)
*   Solve using heuristic or optimization solvers.
